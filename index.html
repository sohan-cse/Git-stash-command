Some commands are

  • git stash save "message" - Saves the current changes with an optional message.
  • git stash list - Lists all stashed changes.
  • git stash apply [stash@{index}] - Applies the specified stash without removing it from the stash list.
  • git stash pop [stash@{index}] - Applies the specified stash and removes it from the stash list.
  • git stash drop [stash@{index}] - Removes the specified stash from the stash list.
  • git stash clear - Removes all stashes.
